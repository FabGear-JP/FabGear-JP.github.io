# FabGear Portfolio

FabGear全体のポートフォリオサイト（GitHub Pages）

## 構成

- `index.html` - トップページ
- `style.css` - スタイルシート
- `docokawa/` - ドコカワLPへのリンク（サブディレクトリ）

## デプロイ

GitHub Pagesで自動公開されます。

URL: https://fabgear-jp.github.io/

## デザイン方針

- ドコカワLPと統一感のある配色（#1a1a2e + #e67e22）
- 個人開発者感を出す（企業サイト感を出さない）
- シンプル・素っ気ない表現
- 煽り文句・マーケティング調を避ける
